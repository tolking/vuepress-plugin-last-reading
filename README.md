# vuepress-plugin-last-reading [WIP]

> record last reading

## Installation

``` sh
yarn add vuepress-plugin-last-reading
# or
npm i vuepress-plugin-last-reading
```

## Usage

``` js
module.exports = {
  plugins: [
    'last-reading'
  ]
}
```

## Options

### popupComponent
- Type: `string`
- Required: `false`

### updatePopup
- Type: `Object`
- Required: `false`
